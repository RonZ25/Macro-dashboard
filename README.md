# Macro Dashboard — CPI (YoY), Unemployment, Real Rates

This Streamlit app shows:
- CPI YoY (computed from **CPIAUCSL**)
- Unemployment rate (**UNRATE**)
- 10‑year real yield (**DFII10** monthly average)

**Data source:** Federal Reserve Economic Data (FRED).  
- CPIAUCSL (CPI index) → https://fred.stlouisfed.org/series/CPIAUCSL  
- UNRATE (Unemployment) → https://fred.stlouisfed.org/series/UNRATE  
- DFII10 (10‑year TIPS real yield) → https://fred.stlouisfed.org/series/DFII10  
- FRED API (series/observations) → https://fred.stlouisfed.org/docs/api/fred/series_observations.html  
- API key (free) → https://fred.stlouisfed.org/docs/api/api_key.html
